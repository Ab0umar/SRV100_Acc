# موديل الواجهة والتنقل UI

## الهدف
واجهة Light Mode حديثة بألوان ويب، وتنقل واضح حسب القسم.

## المطلوب
- Light Mode افتراضي.
- Sidebar أو Top Navigation واضح.
- كل موديل له Shortcuts خاصة به.
- Cards للإحصائيات.
- Tables منظمة.
- Filters في Card ثابت أعلى النتائج.

## قواعد التنقل
- عند دخول الحسابات: تظهر اختصارات الحسابات فقط.
- عند دخول الطبي: تظهر اختصارات الطبي فقط.
- لا يتم خلط القوائم.

## ألوان مقترحة
- Background: فاتح جدًا.
- Cards: أبيض.
- Primary: أزرق أو Indigo.
- Success: أخضر للإيرادات/الصافي.
- Warning: برتقالي للتنبيهات.
- Danger: أحمر للحذف/الإلغاء.

## UX
- الفلاتر واضحة ومختصرة.
- زر بحث واضح.
- زر Reset للفلاتر.
- إجماليات ظاهرة بدون Scroll قدر الإمكان.
