# Agent: QA & Testing

## مهمته
اختبار النظام بعد كل مرحلة.

## اختبارات الحسابات
- فلتر تاريخ فقط.
- فلتر وردية فقط.
- فلتر خدمة فقط.
- فلتر قسم فقط.
- فلتر تاريخ + وردية + خدمة.
- تقرير إيراد الخدمات بدون طبيب.
- استعلام الإيصالات بطبيب.
- حساب مريض بكود.

## اختبارات الواجهة
- دخول الحسابات يظهر شورت كت الحسابات فقط.
- دخول الطبي يظهر شورت كت الطبي فقط.
- Light Mode شغال.
- الفلاتر لا تكسر الشاشة.
- الجداول تظهر أرقام صحيحة.

## اختبارات الطباعة
- معاينة قبل الطباعة.
- نفس الفلاتر.
- نفس الإجماليات.
- شكل مقروء.

## شرط القبول النهائي
النظام يعتبر جاهز لما:
1. كل تقرير يطلع أرقام صحيحة.
2. كل موديل يظهر شورت كت خاص به فقط.
3. الطباعة مطابقة للشاشة.
4. لا يوجد خلط بين الطبي والحسابات.
