# Architectural Decision Records (ADRs)

This directory contains lightweight technical documents that describe an important architectural decision made for the SRV100 project. Each ADR describes the context of the decision, the decision itself, and the consequences of that decision, including trade-offs and alternatives considered.

ADRs are immutable. If a decision is changed, a new ADR is created to supersede the old one.
