# Product

## Register

product

## Users

Clinic reception and admin staff on shared Windows workstations, under bright fluorescent office lighting. They open SELRS Desktop at the start of a shift, pick a connection once, and then live entirely inside the web app. The shell is seen for under 10 seconds per session.

## Product Purpose

SELRS Desktop is a WebView2 shell that wraps the SELRS medical records web application as a standalone Windows program, replacing a browser tab with a focused, installable window. Success means the shell is invisible: it opens fast, connects correctly, and gets out of the way.

## Brand Personality

Functional, precise, unobtrusive.

## Anti-references

- Consumer-app brightness: Fluent white backgrounds with large color blobs
- Enterprise-grey blandness: default WinForms grey with no design intent
- Marketing or product-landing aesthetics: this is a utility, not a storefront

## Design Principles

1. Get out of the way — every shell surface should resolve in under one interaction and disappear.
2. Speak the user's language — Arabic RTL for all user-facing text; English only for technical strings (URLs, code).
3. One accent, used precisely — blue only on the single primary action per surface.
4. Fail clearly — when the server is unreachable, the error page must tell the user exactly what to check, in Arabic.
5. Windows-native feel — spacing, font sizes, and interaction patterns should feel at home on a Windows 11 workstation.

## Accessibility & Inclusion

WCAG AA minimum. Full keyboard navigation (Tab, Enter, Escape) on all dialogs. No motion required for functionality.
